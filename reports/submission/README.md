# Submission write-up

The hackathon submission report lives outside Git as a shared Google Doc. This repository is linked from
that document as the code, compact data-summary, and provenance companion.

The earlier LaTeX/PDF draft in this directory was temporary and became stale after the 100M run, control
LLC trajectories, and localization check. It has been removed from the tracked tree to avoid presenting an
outdated report to reviewers.

For the current evidence, read:

- `README.md` and `PROJECT_STATUS.md` at the repository root;
- `reports/control_comparison/REPORT.md` for the three-condition LLC contrast and localization check;
- `reports/seed_a_llc_trajectory/REPORT.md` for the primary seed-A trajectory and measurement-debugging
  arc;
- `results/02_final_training/` and `results/03_llc_campaign/` for compact machine-readable summaries.

Local DOCX/PDF exports and templates from the Google Doc are intentionally ignored unless explicitly
promoted to a final artifact.
