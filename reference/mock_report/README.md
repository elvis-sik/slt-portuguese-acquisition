# Synthetic mock report

Every result in this directory is fabricated for project planning. This was a pre-run visualization tool:
it helped make the research target concrete by showing what the project would look like if the idea
worked. It is retained for transparency and design history, not as evidence.

The package includes the six-page PDF, editable DOCX, deterministic fake-data generator, figures, source
CSVs, metadata, and rebuild instructions.

Start with:

- `slt_portuguese_synthetic_mock_submission.pdf`
- `README_BUILD.md`
- `../../docs/09_FAKE_REPORT_WALKTHROUGH.md`

Never cite these numbers as empirical evidence and never copy the synthetic CSVs into `results/`. Current
empirical summaries live in `../../reports/seed_a_llc_trajectory/`,
`../../reports/control_comparison/`, and `../../results/`.
